|总字数:760|阅读时长:3分钟|浏览量:

该文章需要一定编程语言基础

基础语法

变量与常量

变量

java类似c语言，都是静态类型语言，使用变量必须要先声明类型

数组

java 数组是固定长度的，声明方式类似c语言

字典

java 集合类似 Python 的列表/字典

常量

java用 final 声明常量

c语言用 const 声明变量

控制结构

条件语句（if-else）

  • java使用 if（...）{...}elif(...){...}else{...} 的结构
  • python：冒号+缩进
  • c语言：与java语法基本相同

循环结构

  • for 循环
  • while 循环

函数

java 方法必须属于某个类，且需声明返回类型和参数类型

控制台输入输出

控制台输入

Java 使用 Scanner 类
导入Scanner库 —> 创建Scanner对象 —> 使用变量指定类型存储数据

控制台输出

Java 使用 System.out 对象

类与对象

类与对象

java 是纯面向对象语言，所有代码必须写在类中

继承

Java 单继承，但支持多接口(当多接口时，不可使用父类中相同的方法)
